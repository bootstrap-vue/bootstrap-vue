# Grid System

> Use our powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes.

```html
<div class="container bv-example-row">
    <div class="row">
        <b-col>1 of 2</b-col>
        <b-col>2 of 2</b-col>
    </div>
    <div class="row">
        <b-col>1 of 3</b-col>
        <b-col>2 of 3</b-col>
        <b-col>3 of 3</b-col>
    </div>
</div>

<!-- col-1.vue -->
```

```html
<div class="container bv-example-row">
    <div class="row">
        <b-col>Column</b-col>
        <b-col>Column</b-col>
        <div class="w-100"></div>
        <b-col>Column</b-col>

</div>
```
